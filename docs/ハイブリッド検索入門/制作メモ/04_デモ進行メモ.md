# デモシナリオ

詳細版は `../03_デモシナリオ.md` を参照。ここでは撮影順だけを残します。

---

## 進行

1. `src/api/schemas.py` を開いて入力項目を確認
2. `src/application/usecases/search_properties.py` を開いて主処理を確認
3. `curl /search` で実行
4. `src/adapters/outbound/ranking/lgbm_reranking_adapter.py` を開く
5. `src/services/ranking/lgbm_reranker.py` を開く
6. `candidate_limit` を変えて差分に触れる

---

## 例コマンド

```bash
curl --get 'http://localhost:8000/search' \
  --data-urlencode 'q=静かな住宅街 2LDK' \
  --data-urlencode 'city=札幌' \
  --data-urlencode 'pet=true' \
  --data-urlencode 'limit=5' \
  --data-urlencode 'candidate_limit=20'
```

---

## 画面で確認するもの

- `me5_score`
- `search_log_id`

- `candidate_limit`
