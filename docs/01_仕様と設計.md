# 不動産検索ランキング基盤（非 Elasticsearch）
## 仕様書・設計書

> 対象：個人開発 / PoC / 小中規模サービス / 学習用ポートフォリオ  
> 前提：Elasticsearch を使わず、軽量・低コスト・高速開発を重視

---

# 1. 本資料の目的

不動産検索ドメインに対し、以下を満たす **非 Elasticsearch 構成の検索ランキング基盤** を定義する。

- 高速な物件検索
- 条件絞り込み（家賃・駅近・築年数など）
- 意味検索（ME5）
- LightGBM によるランキング最適化
- CTR / CVR / 問い合わせ率改善
- Docker / GCP / AWS 展開可能
- 後日 EC検索へ横展開可能

---

# 2. 推奨構成

```text
FastAPI
PostgreSQL
Meilisearch
LightGBM Ranker
ME5 Embedding
Redis（任意）
Docker Compose
````

---

# 3. 技術選定

| レイヤ       | 技術                    | 理由            |
| --------- | --------------------- | ------------- |
| API       | FastAPI               | Python資産活用、高速 |
| DB        | PostgreSQL            | 安定・集計・拡張性     |
| 検索        | Meilisearch           | 軽量・高速・簡単      |
| Ranking   | LightGBM              | 学習ランキングに強い    |
| Embedding | multilingual-e5-large | 日本語意味検索対応     |
| Cache     | Redis                 | 人気検索キャッシュ     |
| Infra     | Docker                | 即起動可能         |

---

# 5. 各技術の役割

## 5.1 Meilisearch

担当：

* エリア検索
* 家賃帯検索
* 駅徒歩検索
* 間取り検索
* ペット可検索
* フリーワード検索

例：

```text
札幌 2LDK 駅近 ペット可
```

役割：

> 高速候補抽出エンジン

---

## 5.2 PostgreSQL

担当：

* 物件マスタ
* 画像情報
* 行動ログ
* CTR集計
* 問い合わせ率
* 特徴量保存

役割：

> 正式データソース

---

## 5.3 ME5

担当：

* 曖昧検索対応
* 類似物件検索
* 説明文意味理解

例：

```text
静かな住宅街
広めの部屋
駅近
住みやすい
```

役割：

> キーワードで拾えない意味検索補完

---

## 5.4 LightGBM Ranker

担当：

候補100件を最終順位化

重視：

* 類似度
* 人気度
* 問い合わせ率
* 家賃妥当性
* 写真品質
* 駅人気

役割：

> CV最大化順位制御

# 追加要件：pgAdmin 導入

PostgreSQL の管理・閲覧・SQL実行を効率化するため、  
Docker 構成へ **pgAdmin** を追加します。

---

# docker-compose.yml（更新版）

```yaml
version: "3.9"

services:

  api:
    build: .
    container_name: real-estate-api
    ports:
      - "8000:8000"
    depends_on:
      - postgres
      - meilisearch

  postgres:
    image: postgres:16
    container_name: real-estate-postgres
    restart: always
    environment:
      POSTGRES_DB: estate
      POSTGRES_USER: admin
      POSTGRES_PASSWORD: password
    ports:
      - "5432:5432"
    volumes:
      - postgres_data:/var/lib/postgresql/data

  pgadmin:
    image: dpage/pgadmin4
    container_name: real-estate-pgadmin
    restart: always
    environment:
      PGADMIN_DEFAULT_EMAIL: admin@example.com
      PGADMIN_DEFAULT_PASSWORD: password
    ports:
      - "5050:80"
    depends_on:
      - postgres

  meilisearch:
    image: getmeili/meilisearch:v1.7
    container_name: real-estate-search
    restart: always
    environment:
      MEILI_NO_ANALYTICS: "true"
    ports:
      - "7700:7700"

  redis:
    image: redis:7
    container_name: real-estate-redis
    restart: always
    ports:
      - "6379:6379"

volumes:
  postgres_data:
````
