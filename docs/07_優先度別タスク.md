**ファイル**: `src/api/routes/search.py`, `src/api/routes/feedback.py`

```python
# 作成予定:
# - SearchParams モデル (q, city, layout, limit, candidate_limit)
# - FeedbackRequest モデル (action, property_id, search_log_id)

class SearchParams(BaseModel):
    q: str = Field(default="", max_length=500)
    city: str | None = Field(default=None, max_length=100)
    layout: str | None = Field(default=None)
    limit: int = Field(default=20, ge=1, le=100)
    candidate_limit: int = Field(default=100, ge=1, le=200)

class FeedbackRequest(BaseModel):
    action: Literal["click", "favorite", "inquiry"]
    property_id: int = Field(gt=0)
    search_log_id: int | None = Field(default=None, gt=0)
```

**期待効果**:
- クエリパラメータの自動バリデーション
- OpenAPI ドキュメント自動生成
- 不正な入力値の早期検出
